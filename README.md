# Gaurang Patil

A high-end, minimalist portfolio designed to showcase engineering work and personal projects with fluid interactions and modern aesthetics.

**[🚀 View Live Demo](https://portfolio-two-peach-67.vercel.app)**

---

## ✦ Features

- **Responsive Design:** A seamless viewing experience across mobile, tablet, and desktop devices.
- **Project Showcase:** Interactive, dynamic cards with premium hover effects.
- **Optimized Performance:** Blazing fast load times and efficient rendering.

## ✦ Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

---

## ✦ Local Development

**1. Clone the repository**
```bash
git clone https://github.com/gaurangpatil97/portfolio.git
cd portfolio
```

**2. Install dependencies**
```bash
cd frontend
npm install
```

**3. Run the development server**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
